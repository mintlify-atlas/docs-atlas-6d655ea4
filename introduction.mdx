## What is Ubik?

Ubik is a full-stack SaaS platform designed to streamline motel and hotel management in the Colombian market. Built with a reactive microservices architecture, Ubik enables:

- **Property owners** to register and manage their establishments, rooms, and services
- **Guests** to discover available motels, check room availability, and make reservations
- **Administrators** to review and approve motel listings through a structured workflow

## Platform overview

Ubik Backend is composed of five reactive microservices, each responsible for a specific domain:

  <Card title="User Management" icon="user" href="/features/user-management">

## Key features

<AccordionGroup>
  <Accordion title="Reactive architecture">
    Built with **Spring WebFlux** and **R2DBC** for non-blocking I/O throughout the entire stack. All service calls return `Mono` or `Flux` reactive types, enabling high throughput with minimal thread usage.
  </Accordion>
  <Accordion title="JWT-based authentication with RBAC">
    The API Gateway validates JWT tokens on every request and injects user identity (`X-User-Id`, `X-User-Username`, `X-User-Role`) as headers into downstream services. Three roles are supported: **Admin**, **Property Owner**, and **User (Client)**.
  </Accordion>
  <Accordion title="Motel approval workflow">
    New motel listings go through a structured approval process: `PENDING` → `UNDER_REVIEW` → `APPROVED` (or `REJECTED`). Admins must verify legal documentation (RUES, RNT, owner documents) before approval.
  </Accordion>
  <Accordion title="MercadoPago payment integration">
    Property owners connect their MercadoPago accounts via OAuth. Guests pay through MercadoPago's hosted checkout, and the platform handles webhook notifications to confirm or reject payments automatically.
  </Accordion>
  <Accordion title="Cloudinary image management">
    Motel and room images are uploaded directly to Cloudinary, with URLs stored in the database. The image upload controller handles multipart form data reactively.
  </Accordion>
  <Accordion title="Angular PWA frontend">
    The companion `ubik-app` is a Progressive Web App built with Angular 20, featuring server-side rendering, Tailwind CSS, PrimeNG components, and Leaflet map integration for geolocation.
  </Accordion>
</AccordionGroup>

## Technology stack

| Layer | Technology |
|-------|-----------|
| Language | Java 17 |
| Framework | Spring Boot 3.2, Spring WebFlux, Spring Cloud Gateway |
| Database | MySQL (Azure Flexible Server) via R2DBC |
| Authentication | JWT (JJWT library) |
| Payments | MercadoPago SDK |
| Image storage | Cloudinary |
| Email | Spring Mail (SMTP) |
| Containerization | Docker, Docker Compose |
| Frontend | Angular 20, TypeScript, Tailwind CSS, PrimeNG |

## Project structure

```
ubik-backend/
├── microservicios/
```
