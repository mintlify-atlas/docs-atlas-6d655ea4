---
title: "Reset password"
description: "Request a password reset token by email, then apply a new password using that token."
---

Password reset is a two-step flow:

1. **Request a reset token** — send the account email to receive a time-limited token by email.
2. **Apply the new password** — submit the token together with the new password to complete the reset.

<Tip>
  The reset token expires after **1 hour**. If it expires before you use it, restart the flow from step 1.
</Tip>

---

## Step 1 — Request a reset token

`POST /api/auth/reset-password-request`

Looks up the account by email, generates a UUID reset token, stores it with a 1-hour expiry, and sends the token to the provided email address.

### Query parameter

<ParamField query="email" type="string" required>
  Email address of the account to reset. Must match an existing user record.
</ParamField>

### Response

#### 200 OK

Returns the generated reset token as a plain string.

<ResponseField name="(body)" type="string" required>
  The UUID reset token. This value is also sent to the account's email address.
</ResponseField>

```json 200
"a3f1b2c4-d5e6-7890-abcd-ef1234567890"
```

#### Error responses

| Status | Cause |
|--------|-------|
| `400 Bad Request` | No account is registered with the given email address. |

### Example

<CodeGroup>

```bash cURL
curl --request POST \
  --url 'https://api.example.com/api/auth/reset-password-request?email=jdoe%40example.com'
```

</CodeGroup>

---

## Step 2 — Apply the new password

`POST /api/auth/reset-password`

Verifies that the token is valid and has not expired, then updates the account password. After a successful reset, the token is invalidated and cannot be reused.

### Request body

<ParamField body="token" type="string" required>
  The reset token received by email in step 1. Cannot be blank.
</ParamField>

<ParamField body="newPassword" type="string" required>
  The new password to set for the account. Cannot be blank.
</ParamField>

### Response

#### 200 OK

<ResponseField name="(body)" type="string" required>
  Confirmation message.
</ResponseField>

```json 200
"Password reset successfully"
```

#### Error responses

| Status | Cause |
|--------|-------|
| `400 Bad Request` | A required field is blank. |
| `400 Bad Request` | The token does not match any account, or the token has expired (older than 1 hour). |

<Warning>
  An expired token returns the same `400` as an unknown token. Request a new token and complete the reset within 1 hour.
</Warning>

### Example

<CodeGroup>

```bash cURL
curl --request POST \
  --url https://api.example.com/api/auth/reset-password \
  --header 'Content-Type: application/json' \
  --data '{
    "token": "a3f1b2c4-d5e6-7890-abcd-ef1234567890",
    "newPassword": "n3wS3cur3P@ss"
  }'
```

</CodeGroup>
